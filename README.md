LocalBrowser
===========================

本地的网页浏览器。可以方便地浏览手机里缓存的网页。也可浏览/缓存在线网页。
该软件开发的初衷是因为作者有次在手机中保存了很多离线网页,但是要打开浏览比较麻烦,为了方便浏览/管理就写了个界面，然后不断尝试和修改中就成这样了，虽然它依然很简单。
